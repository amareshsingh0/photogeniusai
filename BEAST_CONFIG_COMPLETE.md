## Integration Guide

### Python (FastAPI)

1. Create `apps/api/app/config/loader.py`:
```python
import json
from pathlib import Path
from typing import Dict, Any

class BeastConfig:
    _instance = None
    _configs: Dict[str, Any] = {}

    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
            cls._load_all()
        return cls._instance

    @classmethod
    def _load_all(cls):
        config_dir = Path(__file__).parent
        for filename in ["beast_standards.json", "aesthetic_codes.json", ...]:
            with open(config_dir / filename) as f:
                cls._configs[filename.replace(".json", "")] = json.load(f)

    @classmethod
    def get(cls, config_name: str) -> Dict[str, Any]:
        return cls._configs.get(config_name, {})

config = BeastConfig()
```

2. Use in agents:
```python
from app.config.loader import config

# Triage Agent
platform_rules = config.get("platform_contracts")["platforms"]["instagram_feed"]
attention_window = platform_rules["viewer_behavior"]["attention_window_seconds"]  # 1.5

# Creative Director

# Quality Critic
```
