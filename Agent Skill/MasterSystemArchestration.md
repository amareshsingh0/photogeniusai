## Pipeline Execution Protocol

### Phase 0: Intake (Triage Agent)
- Parse request in full
- Detect: asset type, platform, brand, cultural moment, deadline urgency
- Pull brand intelligence if brand is known
- Route to correct fast-path or full pipeline

### Phase 1: Strategy (Creative Director + Brand Intel)
- Creative Director sets the CONCEPT — one sentence that the entire visual must serve
- Brand Intel locks the palette, typography, equity constraints
- These two outputs are NON-NEGOTIABLE for all downstream agents

### Phase 2: Direction (Design Director)
- Receives CD concept + brand palette
- Issues Visual System Decree: composition law, grid, type scale, color usage rules
- This decree is the law. No agent may violate it.

### Phase 3: Production (Copywriter + Senior Designer + Motion Designer)
- Run IN PARALLEL for speed
- Each agent outputs their component
- Copywriter → headlines + platform copy + CTA
- Senior Designer → 3 layout variants (safe / bold / disruptive)
- Motion Designer → animation brief + kinetic notes

### Phase 4: Translation (Prompt Engineers)
- Takes ALL Phase 3 outputs
- Selects optimal model(s) for the asset
- Engineers model-specific prompts with full precision
- Outputs: ready-to-run JSON for each model variant

### Phase 5: Quality Gate (Quality Critic)
- Runs 12-dimension review
- Must score ≥ 8.5/10 overall
- Any dimension scoring < 7 = automatic reject + revision loop
- Outputs: APPROVED / REVISE / ESCALATE

### Phase 6: Learning (Learning Engine)
- Logs what was built, why decisions were made
- Tracks model output quality for continuous improvement
- Feeds back to improve future Triage + Prompt Engineer decisions

## Cultural Intelligence Layer

Every agent in this system operates with a shared Cultural Intelligence Layer.
This layer encodes:

**Current aesthetic zeitgeist (2025-2026):**
- Brutalism × Luxury — raw materials, premium execution
- AI-native aesthetics — procedural patterns, generative textures
- Post-ironic sincerity — meaning it while knowing it's constructed
- Retro-futures — Y2K, '90s rave, '70s sci-fi remixed with today's tech
- Bio-organic geometry — shapes that feel grown, not designed
- Quiet luxury gone loud — understated until it isn't
- Cultural specificity over universalism — hyper-local > generic global

**Generational signal decoding:**
- Gen Z: texture, noise, lo-fi authenticity, anti-polish
- Millennials: clean, purposeful, sustainable-signaling aesthetics
- Gen Alpha: dimensional, interactive-feeling, maximalist but curated
- Crossover: movement-implied statics, depth through layering

**Platform aesthetic contracts:**
- TikTok: Vertical, text-forward, attention in 0.5s, meme-fluent
- Instagram: Square/portrait, editorial quality, color story
- YouTube: Face + text, emotion-first, thumbnail = scroll-stop
- LinkedIn: Authority signals, restraint, no-try-hard professionalism
- Pinterest: Aspirational, high-craft, lifestyle integration
- OOH/Billboard: Single idea, readable at 80mph, 3 words max
- Print: CMYK precision, tactile suggestion, permanence aesthetic

## The Beast Standards

An asset clears "Beast Standard" when it passes ALL of these:

1. **The Stranger Test** — A stranger with zero context understands the core message in 1.5 seconds
2. **The Scroll-Stop Test** — In a feed of 100 posts, this one stops the thumb
3. **The Remove-Color Test** — The composition works in pure black and white
4. **The 10% Size Test** — The design still communicates at 10% of intended size
5. **The Tomorrow Test** — Does this look current, or does it feel dated within 6 months?
6. **The Brand-Remove Test** — Remove the logo. Does it still feel like the brand?
7. **The Emotion Test** — Can you name the SINGLE emotion this triggers in 2 words?
8. **The Competitor Test** — Does this look better than what the top 3 competitors produce?
9. **The Context Test** — Does this fit where it will live (feed, wall, screen, print)?
10. **The Memory Test** — 24 hours later, can someone describe this from memory?

If it fails any one: revise.

## Agent Communication Protocol

Agents communicate via structured JSON handoffs. No free-form text between agents.

```json
{
  "from": "agent_name",
  "to": "agent_name",
  "phase": 1,
  "timestamp": "ISO-8601",
  "locked_decisions": {
    "concept": "...",
    "emotion": "...",
    "palette": []
  },
  "open_decisions": {
    "layout": "pending",
    "copy_variant": "pending"
  },
  "constraints": [],
  "quality_flags": []
}
```

Locked decisions cannot be overridden by downstream agents.
Open decisions are solved by the receiving agent.
Constraints are hard rules from upstream.
Quality flags are warnings to address.

---

## Revision Protocol

When Quality Critic triggers revision:
- Identify the lowest-scoring dimension
- Route ONLY that dimension back to the responsible agent
- Do NOT restart the full pipeline
- Maximum 3 revision cycles before human escalation
