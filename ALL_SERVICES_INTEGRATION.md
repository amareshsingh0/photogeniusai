### Step 2: Fix Imports

Each service needs import fixes:
```python
# Old import (ai-pipeline)

# New import (FastAPI)

```

### Step 3: Add Dependencies

```bash
cd apps/api
pip install -r requirements.txt

# Additional dependencies needed
pip install controlnet_aux
pip install insightface
pip install onnxruntime
```

### Step 4: Create API Endpoints

```python
# apps/api/app/api/v2/generation.py

from app.services.generation import (
    generation_service,
    quality_scorer,
    orchestrator,
    two_pass_generator
)

@router.post("/two-pass")
async def generate_two_pass(request: GenerateRequest):
    """Generate with preview + full quality"""
    result = await two_pass_generator.generate(
        prompt=request.prompt,
        mode=request.mode
    )
    return result
```
